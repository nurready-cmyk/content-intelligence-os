## PART 1: Видение и философия

### Project Name
**Content Intelligence OS**

### Vision
Создать интеллектуальную систему мирового уровня, которая помогает предпринимателю:
- строить сильный личный бренд
- создавать высокоценный контент
- привлекать клиентов на консалтинг
- становиться признанным экспертом
- формировать долгосрочный капитал доверия

Через YouTube, подкасты, выступления и профессиональный контент.

Система работает как **виртуальный аналитический отдел, исследовательский центр, контент-стратег, редактор, сценарист и совет директоров одновременно.**

> ❌ Система не должна быть очередным Telegram-ботом для поиска трендов.

### Главный вопрос системы

> *«Если сегодня можно создать только один контент-продукт — какой контент даст максимальный рост авторитета, доверия, влияния и будущего дохода?»*

### Long-Term Mission

Помочь пользователю построить **международный экспертный бренд.**

Целевые результаты:
- Сильный YouTube-канал на русском языке
- Сильный YouTube-канал на английском языке
- Регулярные приглашения на форумы
- Высокий уровень доверия аудитории
- Поток клиентов на консалтинг
- Развитие интеллектуального капитала
- Создание собственной авторской методологии
- Признание как эксперта мирового уровня

### Core Philosophy

Система балансирует между:

## PART 2: Источники данных и движки

### Overview

Content Intelligence OS функционирует как **система стратегической разведки контента.**

Главная задача: не находить популярный контент, а **выявлять закономерности**, которые объясняют почему контент становится популярным, какие потребности он закрывает и какие темы будут востребованы в будущем.

### Источники данных

**Level 1 — Обязательные:**
- YouTube (список каналов, задаваемый пользователем)
- Google Trends
- Reddit
- X (Twitter)
- Бизнес-подкасты
- Новостные ресурсы

**Level 2:**
- Научно-популярные публикации
- Исследовательские платформы
- Профессиональные блоги
- Отраслевые отчёты
- Управленческие, экономические, технологические исследования

**Level 3:**
- Истории и кейсы компаний
- Исторические события
- Биографии лидеров
- Книги по бизнесу, психологии, управлению, переговорам

## PART 3: Telegram-интерфейс и отчёты

### Принцип работы

> **Decision First. Data Second.**

Пользователь приходит не за данными. Пользователь приходит за ответом: **«Что делать дальше?»**

### Daily Executive Briefing

Автоматически каждый день в **08:00** (время настраивается).

**Структура ежедневного отчёта:**

| Секция | Содержание |
|---|---|
| **1. Global Intelligence Summary** | 3-5 ключевых наблюдений о рынке знаний |
| **2. Top 10 Global Trends** | Название, скорость роста, причина, аудитория, потенциал |
| **3. Content Opportunities** | Топ-10 идей с болями, желаниями и потенциалом |
| **4. Video of the Day** ⭐ | Одно видео. Только одно. С полным обоснованием |
| **5. Authority Builder** | Темы для построения бренда (не обязательно вирусные) |
| **6. Consulting Opportunities** | Темы с высоким потенциалом привлечения клиентов |
| **7. Forum Speaker Insights** | Темы для форумов, конференций, подкастов |
| **8. English Channel Opportunities** | Отдельный блок для англоязычного канала |

Каждый отчёт завершается блоком **RECOMMENDED ACTION:**
- Что сделать сегодня
- Что сделать на этой неделе
- Что сделать в этом месяце

### Периодические отчёты

- **Weekly** (каждое воскресенье): тренды недели, новые каналы, изменение интересов, прогноз
- **Monthly**: что растёт, что теряет актуальность, новые возможности

## PART 4: Система генерации сценариев

### Core Principle

> Система никогда не создаёт контент по принципу: «Вот несколько советов.»

Каждый сценарий отвечает:
- Почему это происходит?
- Какая закономерность стоит за этим?
- Какой закон нарушается?
- Что будет дальше?
- Что делать зрителю?

### Стандарты качества

**Уровень анализа:** McKinsey / Harvard Business School / Bain / BCG

**Запрещено:**
- Поверхностное обучение
- Мотивационные лозунги
- Банальные советы
- Неподтверждённые утверждения
- Контент без логики

### Поддерживаемые форматы

Long-form YouTube, Shorts, Подкасты, Выступления, Интервью, Обучающие видео, Разбор кейсов, Аналитические выпуски.

### Hook Generation Engine

Для каждой темы — минимум **20 вариантов хуков.**

Типы хуков: парадокс, неожиданный факт, ошибка большинства, разрушение мифа, угроза, потеря, история, конфликт, сильный вопрос, контринтуитивное утверждение.

**Пример:**

> ❌ Плохо: *«Сегодня поговорим о прибыли.»*
>
> ✅ Хорошо: *«Большинство предпринимателей уверены, что знают свою прибыль. Но именно эта уверенность часто приводит бизнес к проблемам.»*

### Структура Long-form видео

```
Hook → Большой вопрос → Почему это важно → История →
Корневая причина → Закон или принцип → Исследование →
Кейс → Решение → Практические шаги → Главная мысль → CTA
```

### Структура Shorts

```
Hook → Проблема → Неожиданный инсайт → Решение → Запоминающаяся мысль
```

### Output сценария

Каждый готовый сценарий содержит:

```
Название
Подзаголовок
Цель видео
Целевая аудитория
Главная боль
Главное желание
Hook (топ-3 из 20)
Ключевой конфликт
Главная закономерность
Закон
История / исторический пример
Исследование / статистика
Кейс компании
Основные тезисы
Практические шаги
CTA
Score по 8 метрикам
Финальная рекомендация: почему снимать именно сейчас
```

## MVP-архитектура

> Специальный раздел: реалистичный план для запуска **без опыта программирования за 7–14 дней.**

### Что такое MVP в данном проекте

MVP — это **не бот и не сервер.** MVP — это мощная промпт-система внутри Claude.ai, которую вы открываете каждое утро и получаете полный отчёт за 2 минуты.

### Из чего состоит MVP

**6 промптов, покрывающих весь PRD:**

| Промпт | Функция |
|---|---|
| **MASTER SYSTEM PROMPT** | Ядро системы: личность, философия, контекст о пользователе |
| **DAILY BRIEFING** | Ежедневный отчёт по всем 8 секциям |
| **CHANNEL ANALYSIS** | Глубокий анализ конкретного канала из вашего списка |
| **IDEA SCORING** | Оценка идеи по всем 8 метрикам |
| **SCRIPT GENERATOR** | Полный сценарий с хуками, кейсами, законами |
| **WEEKLY STRATEGY** | Еженедельный стратегический обзор |

### Что нужно сделать один раз

1. Зарегистрироваться в Google Cloud и получить **YouTube Data API ключ** (бесплатно, 10 минут)
2. Составить список **10–30 YouTube-каналов** для анализа (конкуренты и эталоны)
3. Создать **Telegram-бота** через @BotFather (2 минуты)
4. Создать **Google Sheets** таблицу для хранения идей и сценариев

### Переход к автоматизации (Phase 2)

Когда промпт-система заработает и докажет ценность — Claude Code пишет Python-скрипт, который:
- Запускается на VPS ($3-5/мес на Hetzner)
- Автоматически собирает данные из YouTube API
- Отправляет отчёт в Telegram каждое утро в 08:00
- Сохраняет всё в Google Sheets

## Правила системы

> Система никогда не заканчивает отчёт просто данными.
> Каждый отчёт обязан завершаться конкретной рекомендацией.
> Система помогает принимать решения, а не просто показывает информацию.

> Система не создаёт контент ради просмотров.
> Система создаёт контент, который помогает человеку мыслить глубже, принимать лучшие решения, усиливает доверие к автору и формирует долгосрочный экспертный бренд.

> Система учится после каждого опубликованного видео, каждого комментария, каждого клиента, каждого успеха и каждой ошибки.
> Система становится умнее вместе с пользователем.
> Именно это является главным конкурентным преимуществом платформы.
